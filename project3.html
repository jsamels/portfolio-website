Adventure Outfitters

View Project

Project Description: I developed a functioning web application for a theoretical outfitting company using HTML, Python, MySQL, Flask, and Bootstrap. The application features an interactive and responsive user interface, allowing users to update a live database through an easy design that simplifies the editing and adding of people and equipment to the database. Additionally, there is a feature to rent equipment using datetime functions to check out and return equipment.

Equipment Management: Manage and update equipment inventory.

Home Screen: User-friendly interface for easy navigation.

People Management: Manage and update user information.
